caper.js


caper.js is a simple tool for creative fun collaboration. Originally an experiment with HTML5 canvas, caper.js is built for the co-creation of low-fidelity designs in realtime with others across the Web. It's open source on Github.

Full screen mode is also supported

Examples

burb 

$('#caper').caper();

No controls. 

$('#caper').caper({
	enableControls: false
});

Fullscreen 

$('#caper').caper({
	fullScreen: true
});

Can I haz callbacks? Sure. 

$('#caper').caper({
	onDraw: function( everythingYouNeed ){
		// why dont you push everythingYouNeed to other clients, perhaps via Socket.IO
	},
	onWrite: function( everythingYouNeed ){
		// as above ^-^
	},
});

$('#caper').caper('draw', everyThingItNeeds ); // trigger draw event
$('#caper').caper('write', everyThingItNeeds ); // trigger write event

What is this magic?

Lots of JavaScript and HTML5 canvas.
